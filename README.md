# Manuel CSS

## Ajouter du CSS

Il y a trois manières d'ajouter du CSS à un élémenet

* inline
* internal
* external

**Inline **

`<p style="color : red;">Ceci est un paragraphe</p>`

**Internal**

Dans le head juste après la balise title

```
<style>
  p {
    colore : red;
  }
</style>
```

**External**

Dans le head juste après la balise title

```
<link href='./style.css' rel='stylesheet'>
```
